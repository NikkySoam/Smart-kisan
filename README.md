# 🌾 Smart Kisan

Smart Kisan is a full-stack agriculture management platform built to help farmers digitally manage their farming activities, expenses, and crop health.

The idea for this project originated from a real-world problem faced in my own family farm, where tracking water usage, fertilizer expenses, labour costs, and field records manually was difficult and time-consuming. Smart Kisan aims to simplify farm management through technology and data-driven insights.

## 🚀 Features

### 🌱 Farm Management

* Manage multiple fields
* Track field-wise farming activities
* Maintain organized farm records

### 💧 Water Management

* Record tubewell water usage
* Calculate irrigation costs automatically
* Monitor water consumption history

### 🌿 Fertilizer Management

* Track fertilizer usage
* Maintain expense records
* View fertilizer history

### 👨‍🌾 Labour & Equipment Management

* Record labour costs
* Manage equipment expenses
* Track operational farming costs

### 🤖 AI Crop Doctor

* Upload crop images
* AI-powered disease and pest detection
* Treatment recommendations
* Preventive farming advice
* Crop scan history

### 📊 Reports & Analytics

* Generate detailed farming reports
* Generate bills and expense summaries
* Visualize data using statistical graphs
* Analyze farming expenses and trends

### 🌐 Multilingual Support

* English Language Support
* Hindi Language Support

## ⚙️ Installation

### Clone Repository

```bash
git clone <https://github.com/NikkySoam/Smart-kisan.git>
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

### Backend

```bash
cd backend
npm install
npm run dev
```

### Environment Variables

Create a `.env` file inside the backend directory:

```env
PORT=5000
MONGO_URI=
JWT_SECRET=
GEMINI_API_KEY=
CLOUDINARY_CLOUD_NAME=
CLOUDINARY_API_KEY=
CLOUDINARY_API_SECRET=
```
